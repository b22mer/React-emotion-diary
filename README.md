# React emotion-diary-app

- React를 활용한 감정 일기장
- localstorage를 활용한 DB 설정
- firebase를 활용한 배포
- 불필요한 화면 렌더링 최적화
- Open Graph 설정

## 구동환경

- VSCODE - create-react-app
- node.js v 18.12.1
- firebase
- npm start (localhost: 3000)

## 프로젝트 기능

1. 일기장 CRUD
2. 일기장 시간순, 감정별 정렬
3. 사용자 친화적 반응 처리
4. Localstoage를 이용한 일기장 DB
5. 화면 렌더링 최적화
